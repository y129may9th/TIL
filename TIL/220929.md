## VS code に移管
### バージョン管理
```json
// setting.json
  "dart.flutterSdkPaths": [
    "/Users/<user>/.asdf/installs/flutter/",
  ],
```
```bash
asdf local flutter 3.3.2-stable
git init
```
- `.tool-version` や `.git` ファイルができる
