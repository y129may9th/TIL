# JSON について
- JavaScript Object Notation
- JSON データへアクセスしたい場合は、JavaScript オブジェクトへ変換する必要がある。

## JSONが対応しているデータ型
文字列、数値、配列、論理型やその他のリテラル型

## 注意点
- JSON では文字列とプロパティ名を `""` で括る必要があり、`''`は、JSON 文字列全体を囲む以外では無効。
- [JSONLint](https://jsonlint.com/) などで妥当性を確認できる。
