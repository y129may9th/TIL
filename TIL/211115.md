# CSS
### `vertical-align: middle;` でもリストアイテムの marker が下にずれる
-> middleが指定された場合、baselineよりもx-heightの高さの半分だけ上の位置に、画像の中央位置が揃うので、日本語だと、merker が下にずれていみえる
> 日本語の文字というのは、アルファベットの文字と比較すると、行のやや上寄りに寄っているように見える
