# PostCSS
- Node.js 製の CSS ツールを作るためのフレームワーク
- CSS パーサー

## PostCSS 製のツール
- 「Autoprefixer」ベンダープリフィックスを自動で付与する
- 「cssnext」未来のCSSの構文の一部を今のブラウザで解釈できるようにする
- 「stylelint」CSSリンター

```
sassファイル
↓ pre processor
cssファイル
↓ post processor
変換後のcssファイル
```
