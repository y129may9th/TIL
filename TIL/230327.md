# Shared preferences plugin
ドキュメント: https://pub.dev/packages/shared_preferences

## Valueで使用可能な型
```
- int型
- double型
- bool型
- String型
- String型のリスト
```

## 操作
- 削除
```dart
  // key を指定して削除
  await prefs.remove("keys");

  // すべての値をクリアする
  final bool ok = await prefs.clear();
```
