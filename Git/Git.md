## Git
1. ローカルファイルにリポジトリ作成  
`$ git init`
1. ステージング  
`$ git add -A`  
1. コミット  
`$ git commit -m "initial commit"`
1. GitHubにリモートリポジトリ作成
1. リモートリポジトリの追加  
`$ git remote add origin https://XXXXX` 
1. ブランチの状態確認  
`$ git status`
1. プッシュ  
`$ git push origin master`


峻輔くんが使っていたコマンド  
`$ git log --oneline`

- リモートリポジトリとローカルリポジトリの差分が一致していなかったため、push する際にreject された。
