## インスタンス変数
オブジェクトが抱える変数  
`@name`

```ruby
Class User 
    # setter
    def name= (name)
        @name = name
    end

    # getter
    def name
        @name
    end
end
```
```
> user = User.new
> user.name= "yui"
=> "yui"
> user.name
=> "yui"
> user
=> #<User:0x00007fea1995e9e8 @name="yui">
```
`user（インスタンス）が @name (インスタンス変数)を抱えている`



## ローカル変数との違い
- ローカル変数  
(ex.)user
メソッド内ので定義すると、そのメソッドの中でしか使えない
- インスタンス変数
(ex.)@name
オブジェクトの中のどのメソッドからも呼び出し可能
```ruby
Class Myclass 
    def method1
        number = 100
    end

    def method2
        number
    end
end
```

```
> object = Myclass.new
> object.method1
=> 100

> object.method2
=> Error
```
method2ではmethod1で定義したローカル変数`number`は使えない
